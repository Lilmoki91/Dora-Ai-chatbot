## 📖 Tentang Projek

**Dora AI Chatbot** adalah sebuah projek sumber terbuka (open-source) yang dibangunkan menggunakan **HTML**, **CSS**, dan **JavaScript**. Ia direka untuk memberikan pengalaman interaktif yang menyeronokkan dengan elemen simulasi quantum deepthink.

> 💡 **Objektif Utama**: Menyediakan platform pembelajaran AI yang mudah diakses oleh semua lapisan masyarakat Malaysia sambil memupuk minat dalam teknologi kecerdasan buatan.

Projek ini dimulakan sebagai inisiatif peribadi untuk menunjukkan bagaimana AI boleh diintegrasikan dalam aplikasi web ringkas tanpa memerlukan infrastruktur yang kompleks.

## 🚀 Mulakan

### Keperluan Asas
- Pelayar web moden (Chrome, Firefox, Edge, Safari)
- Text editor (VS Code, Notepad++, Sublime Text) - untuk ubahsuai
- Pengetahuan asas HTML/CSS/JavaScript (jika nak ubahsuai)

### Cara Pasang (Lokal)

```bash
# Clone repositori ini
git clone https://github.com/Lilmoki91/Dora-Ai-chatbot.git

# Masuk ke folder projek
cd Dora-Ai-chatbot

# Buka index.html dalam pelayar anda
# (Double-click fail index.html atau right-click > Open with browser)
```

Atau download terus:

1. Pergi ke https://github.com/Lilmoki91/Dora-Ai-chatbot
2. Klik butang hijau "Code" ▼
3. Pilih "Download ZIP"
4. Extract ZIP dan buka index.html

---

🎯 Cara Guna

1. Buka aplikasi melalui demo langsung atau fail index.html
2. Taip mesej anda dalam ruangan chat yang disediakan
3. Tekan Enter atau butang hantar (ikon paper plane)
4. Tunggu respons dari Dora AI (akan muncul dalam beberapa saat)
5. Terus berbual - ulang langkah 2-4 untuk interaksi seterusnya

Tips Penggunaan:

· Cuba tanya soalan umum untuk melihat respons AI
· Gunakan bahasa Melayu atau Inggeris
· Refresh halaman jika mahu mulakan sesi baru

## 🤝 Sumbangan

Kami sangat mengalu-alukan sumbangan daripada komuniti! Sama ada anda developer, designer, atau pengguna biasa, sumbangan anda amat dihargai.

Cara Menyumbang

```bash
# 1. Fork repositori ini (klik butang Fork di GitHub)

# 2. Clone fork anda ke komputer
git clone https://github.com/[nama-pengguna-anda]/Dora-Ai-chatbot.git

# 3. Buat branch baru untuk fitur anda
git checkout -b fitur-baru-anda

# 4. Buat perubahan dan commit
git add .
git commit -m "Tambah: Fitur menarik - [huraian ringkas]"

# 5. Push ke branch anda
git push origin fitur-baru-anda

# 6. Buat Pull Request (PR) di GitHub
```

### Apa yang Boleh Disumbangkan?

• 🐛 Lapor bug - Jumpa masalah? Buat issue di GitHub

• 💡 Cadang fitur - Ada idea menarik? Kongsikan!

• 📝 Baiki dokumentasi - Bantu kemas kini README

• 🌐 Terjemahan - Bantu terjemah ke pelbagai bahasa

• 🎨 Reka bentuk UI/UX - Cadangkan penambahbaikan visual

• ⚡ Optimisasi kod - Bantu jadikan kod lebih efisien

Panduan Sumbangan:

• Pastikan kod anda kemas dan mempunyai komen

• Patuhi struktur folder yang sedia ada

• Nyatakan dengan jelas apa yang anda tambah/baiki dalam commit message

• Uji perubahan anda sebelum hantar PR
